Google Search Console API Kurulumu

SEO verilerini görmek için aşağıdaki adımları tamamlayın.

1

Google Cloud Console'da proje oluşturun

console.cloud.google.com adresinden yeni bir proje oluşturun veya mevcut projenizi seçin.

2

Search Console API'yi etkinleştirin

Search Console API ve Search Console API (webmasters) sayfalarından "Enable" butonuna tıklayın.

3

Service Account oluşturun

IAM & Admin > Service Accounts'tan yeni bir service account oluşturun. "Keys" sekmesinden JSON key dosyasını indirin.

4

Service account'u Search Console'a ekleyin

Search Console > Ayarlar > Kullanıcılar ve izinler > "Kullanıcı ekle". Service account e-posta adresini (örn. norya-seo@project.iam.gserviceaccount.com) "Tam" izinle ekleyin.

5

.env dosyasına yol ekleyin

İndirdiğiniz JSON key dosyasını sunucuya yükleyin ve .env dosyanıza ekleyin:

GSC_SERVICE_ACCOUNT_JSON=/path/to/service-account-key.json

Not: Kurulumu tamamladıktan sonra uygulamayı yeniden başlatın. Veriler 5 dakika önbelleğe alınır.

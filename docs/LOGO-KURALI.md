# Norya logo kuralı

**Tüm logolarda olması gereken:** Büyük **N** harfi ile yanında (veya altında/üstünde) **ince dikey çizgi** (#e07a5f turuncu). Bu N + çizgi markanın sabit öğesidir.

- Ana logo: Üstte **N** + ince çizgi, altta **orya** (site renkleri: navy #0a1929, çizgi #e07a5f).
- Yeni logo / ikon tasarlanırken de N ve ince çizgi mutlaka yer almalı.

# Kan Testlerinde Çıkan Sonuçlar — Norya

Norya raporunda **tüm laboratuvarın yazdığı parametreler** (AI çıktısında **Değerler** / **Test sonuçları** bölümünde yer alanlar) kartlara eklenir. Aşağıda sistemin **tanıdığı ve kategorilere ayırdığı** yaygın parametreler özetleniyor.

---

## 1. Hemogram (Tam kan sayımı)

| Parametre | Açıklama / referans (bilgi) |
|-----------|-----------------------------|
| WBC | Lökosit, 4.5–11.0 K/µL |
| RBC | Eritrosit, 4.5–5.5 M/µL (E), 4.0–5.0 (K) |
| HGB / Hemoglobin | 13.5–17.5 g/dL (E), 12.0–15.5 (K) |
| HCT | Hematokrit, %38.8–50 (E), %34.9–44.5 (K) |
| PLT | Trombosit, 150–400 K/µL |
| MCV | Ortalama eritrosit hacmi, 80–100 fL |
| RDW | Eritrosit dağılım genişliği, %11.5–14.5 |
| MPV, PDW, lenfosit vb. | Raporda geçen diğer hemogram parametreleri de kartlara eklenir |

## Raporda nasıl görünür?

- **Biyobelirteç Özeti:** Metabolik, Kardiyovasküler, Karaciğer, Enflamasyon, Demir/Vitamin gibi gruplarda öne çıkan parametreler.
- **Test Sonuçları:** Yukarıdaki kategorilere göre (Hemogram, CRP, Glukoz, Lipidler, Karaciğer, Böbrek, Demir, Tiroid, **Diğer Değerler**) tüm parametreler kartlarda listelenir.
- Laboratuvarda çıkan **her parametre** (AI çıktısında **Değerler** veya **Test sonuçları** bölümünde geçiyorsa) raporda yer alır; tanımlı olmayanlar **Diğer Değerler** altında gösterilir.

Referans aralıkları bilgi amaçlıdır; kesin değerlendirme için laboratuvarın kendi referansları ve hekim yorumu esas alınmalıdır.
